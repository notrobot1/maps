1. Запускаем key.exe. Данный файл ждет сообщение от браузера для набора текста.
	1.1 В файла hello.txt через ; прописаны приветствия. 
	1.2 В файле text.txt через ; содержимое сообщений.
2. Проверяем, что у нас включена английская раскладка. 

3. В GoogleChrome переходим на chrome://extensions/ включаем режим разработчика, нажимаем загрузить упакованное расширение и выбираем пусть до папкм js. Активируем расширение.

4. Переходим по ссылке https://www.grailed.com/designers. Скрипт начинает работу. 

5. Все исходники я залил в закрытый репозиторий https://github.com/notrobot1/chrome_auto. Если есть акк. на гит хабе я открою доступ.
